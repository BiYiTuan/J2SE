--编译
javac Hello.java
如果代码里有包,如: package com; 要先手动建一个com文件夹(如果不手动建立com文件夹，直接编译的话，
编译正常，但执行的时候会出现异常)，编译命令为：java –d . Hello.java

--执行
java Hello
(有包的时候: java com.Hello)

--生成文档(必须有公开类和注释文档)
javadoc –d test Hello.java

--打包
1.编译 javac Hello.java
2.打包 jar cvf t.jar Hello.class
3.运行 java -jar t.jar

--运行jar包
java -classpath ./* com.alibaba.dubbo.container.Main
java -cp ./* com.alibaba.dubbo.container.Main
(-classpath可以简写为-cp，引入jar包不支持*.jar这种通配，会无法匹配，但是*却可以)

--解压war包
jar -xvf game.war

--查看jdk中的2D演示
jdk\demo\jfc\java2D>java -jar Java2Demo.jar

java语言有三种注释
1.单行注释： // 

2.多行注释  
/*
………… 
………… 
*/

3.文档注释 (文件注释，类注释，方法注释)
/**
 * …………………
 * …………………
 */
注:文档注释中支持HTML标记。
