Java 环境变量的配置：
   java_home：D:\jdk
   path：D:\jdk\bin

一.运行java应用程序：
1.写好testHello.java源程序;
2.在“运行”中输入以下命令：
D:\JavaExamples>javac testHello.java
生成testHello.class字节码文件；
3.在“运行”中输入以下命令：
D:\JavaExamples>java testHello

二.运行java小程序：
1.写好Hello.java源程序;
2.在“运行”中输入“D:\JavaExamples>javac Hello.java”
生成Hello.class字节码文件，
3.编写嵌入了Hello.class代码的Hello.html文件
4.在“运行”中输入：D:\JavaExamples>appletviewer Hello.html”
或在IE浏览器中输入：D:\JavaExamples\Hello.html。
